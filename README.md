# StyleNear
StyleNear is a D2C fashion startup with the mission of making affordable, ethnically-inspired fashion accessible to customers in Tier-2 and Tier-3 Indian citie. The brand sells across six product categories: Kurta, Saree, Western Tops, Denim, Ethnic Wear, Accessories, and Footwear, priced between Rs. 449 and Rs. 2,999 MRP.
